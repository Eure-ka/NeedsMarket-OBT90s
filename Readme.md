#  [Spring]게시판 프로젝트
- #### 개요
  - ##### Spring MVC Project를 이용하여 중고나라와 비슷한 중고거래 사이트를 만들고 싶어 구현함
- #### 개발 환경
  - ##### STS 3.9.14
  - ##### JDK 1.8
  - ##### Tomcat 9.0
  - ##### Spring 4.1.1
  - ##### Maven 2.5.1
  - ##### Oracle 11g
- #### 주요기능
  1.회원가입 및 로그인/로그아웃  
  
  2.글 쓰기/수정/삭제/읽기  
  
  3.이미지 올리기/미리보기  
  
  4.댓글 읽기/쓰기/수정/삭제   
  
  5.답글 쓰기/읽기/수정/삭제  
  
  6.좋아요(추천)   
  
  7.조회수  
  
  8.회원정보 수정및 회원 탈퇴
